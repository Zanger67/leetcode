#### [e619.sql](<../my-submissions/e619.sql>)
```SQL
SELECT 
    MAX(num) AS 'num'
FROM (
    SELECT num
    FROM MyNumbers mn
    GROUP BY num
    HAVING COUNT(num) = 1
) AS unique_vals;
```
