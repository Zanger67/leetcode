```Python

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        output = 0
        maxxPrice = prices[-1]
        for price in reversed(prices) :
            if price > maxxPrice :
                maxxPrice = price
            elif maxxPrice - price > output :
                output = maxxPrice - price
        return output

```
