### SQL
#### [e183.sql](<../my-submissions/e183.sql>)
```SQL

SELECT
    name AS 'Customers'
FROM
    Customers c LEFT JOIN Orders o ON c.id = o.customerId
WHERE 
    o.customerId IS NULL;
```
