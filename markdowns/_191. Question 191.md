#### [191.number-of-1-bits.py](<../my-submissions/191.number-of-1-bits.py>)
```Python

class Solution:
    def hammingWeight(self, n: int) -> int:
        counter = 0
        while n > 0 :
            if n % 2 == 1 :
                counter += 1
            n //= 2
        return counter

```
